## 概要

本ドキュメントはTempo AIのAIプロンプト設計に関するインデックスです。

詳細な仕様は以下の3つのドキュメントに分割されています：

---

## ドキュメント構成

### 1. [ai-prompt-core.md](./ai/ai-prompt-core.md) - コア仕様

AIプロンプトの基本構造とシステム設計：

- **概要**: アドバイス生成の基本方針、バランス設計
- **JSON出力スキーマ**: ホーム画面・詳細画面の対応、データ構造
- **プロンプト構成**: 3層構造とPrompt Cachingキャッシュ戦略
- **Layer 1 システムプロンプト**: 役割定義、禁止事項、出力形式
- **トーン・文体ガイドライン**: キャラクター設定、文体ルール

### 2. [ai-prompt-advice.md](./ai/ai-prompt-advice.md) - アドバイス例文集

関心ごと別の例文とペルソナ別サンプル：

- **Layer 2 例文**: 関心ごと別の例文セット
  - 運動・フィットネス向け
  - 美容・スキンケア向け
  - メンタル・ストレス向け
  - エネルギー・パフォーマンス向け
- **ペルソナ別アドバイス例文集**: 代表ペルソナごとのサンプル
- **今日のトライ・今週のトライ**: カテゴリー別の例文パターン

### 3. [ai-prompt-analysis.md](./ai/ai-prompt-analysis.md) - データ分析・インサイト生成

ユーザーデータの送信形式と相関分析活用：

- **Layer 3 ユーザーデータ構造**: プロファイル、健康データ、環境データ
- **メトリクスデータのJSON形式**: 新メトリクス体系（睡眠・HRV・リズム・活動量）
- **相関分析データの活用**: 相関結果の送信フォーマット、プロンプトへの組み込み
- **インサイト生成ルール**: メトリクスステータスに基づくアドバイス方向
- **追加アドバイス用プロンプト**: 昼・夕のフォローアップ

## クイックリファレンス

### JSON出力スキーマ概要

```json
{
  "greeting": "〇〇さん、おはようございます",
  "condition": {
    "summary": "（3-4文）",
    "detail": "（5-8文）"
  },
  "action_suggestions": [
    { "icon": "fitness", "title": "...", "detail": "..." }
  ],
  "closing_message": "（1-2文）",
  "daily_try": {
    "title": "...",
    "summary": "...",
    "detail": "..."
  },
  "weekly_try": null // 月曜のみ値を設定
}
```

### プロンプト3層構造

| レイヤー | 内容 | キャッシュ | 推定トークン |
|---------|------|----------|------------|
| Layer 1 | システムプロンプト | 対象（TTL: 1h） | 約1,500 |
| Layer 2 | 例文（関心ごと別） | 対象（TTL: 5m） | 約2,000 |
| Layer 3 | ユーザーデータ | 対象外 | 約800-1,200 |

### 新メトリクス体系

| メトリクス | 概念 | スコア | ステータス |
|-----------|------|--------|----------|
| 睡眠 | 睡眠の質と量 | 0-100 | 80+: 最高, 60-79: 良好, 40-59: 普通, 20-39: やや低下, 0-19: 要改善 |
| HRV | 心拍変動 | 0-100 | 同上 |
| リズム | サーカディアンリズム | 0-100 | 同上 |
| 活動量 | 身体活動レベル | 0-100 | 同上 |
| ストレスグラフ | 自律神経バランス | -100〜+100 | 正: リラックス, 負: ストレス |
