## 1. Layer 2: 例文（関心ごと別）

関心ごとタグに応じて、以下の例文セットから適切なものを選択してプロンプトに含めます。

### 選択ロジック

```
優先タグ（最初に選択されたタグ）に基づいて例文セットを選択

運動・フィットネス → fitness_examples
美容・スキンケア → beauty_examples
メンタル・ストレス → mental_examples
エネルギー・パフォーマンス → work_examples
栄養・食事 → nutrition_examples
睡眠 → sleep_examples
```
