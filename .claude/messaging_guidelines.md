# Messaging & Copywriting Guidelines for Health & Wellness Apps

このドキュメントは、健康・ウェルネスアプリにおける適切なメッセージング、コピーライティングの原則をまとめたものです。
心理学的根拠に基づき、ユーザーのモチベーションを高め、行動変容を促す言葉選びを実現します。

## 基本原則

### 1. Progress-Focused（進捗重視）

**原則**: 欠点や失敗を指摘するのではなく、進捗と改善に焦点を当てる。

**❌ 避けるべき**：

- 「睡眠時間が不足しています」
- 「目標未達成です」
- 「運動量が少なすぎます」
- 「カロリーオーバーです」

**✅ 推奨**：

- 「今週は平均 6 時間の睡眠でした。7-8 時間を目指してみませんか？」
- 「目標まであと少しです！今日も頑張りましょう」
- 「先週より歩数が増えています。この調子です」
- 「今日のバランスは良好です。明日も続けてみませんか？」

### 2. Suggestion-Based（提案型）

**原則**: 命令や指示ではなく、提案や選択肢の提示。

**❌ 避けるべき**：

- 「〜してください」
- 「〜する必要があります」
- 「〜しなければなりません」
- 「必ず〜してください」

**✅ 推奨**：

- 「〜してみませんか？」
- 「〜がおすすめです」
- 「〜すると良いかもしれません」
- 「〜を試してみるのはいかがでしょうか？」

### 3. Evidence-Backed（根拠提示）

**原則**: アドバイスの根拠を簡潔に説明し、信頼性を高める。

**❌ 避けるべき**：

- 「これが正解です」
- 「絶対に効果があります」
- 「医学的に証明されています」（具体性がない）

**✅ 推奨**：

- 「あなたの睡眠パターンから、〜が効果的かもしれません」
- 「研究によると、〜が推奨されています」
- 「多くの人は〜で改善しています」
- 「あなたのデータを分析した結果、〜が見られました」

### 4. User Empowerment（ユーザーの自律性尊重）

**原則**: ユーザーが自分で選択・判断できることを強調。

**❌ 避けるべき**：

- 「このプランに従ってください」
- 「設定を変更する必要があります」

**✅ 推奨**：

- 「あなたに合ったプランを選んでください」
- 「設定はいつでも変更できます」
- 「あなたのペースで進めましょう」
- 「無理のない範囲でお試しください」

### 5. Non-Medical Disclaimer（医療的助言ではないことの明示）

**原則**: AI アドバイスは医療的助言ではないことを適切に伝える。

**必須表記箇所**：

- 初回起動時のオンボーディング
- 健康アドバイス表示の下部（小さなフォントで OK）
- 設定画面の「このアプリについて」

**推奨表現**：

- 「このアプリは一般的な健康情報を提供するものであり、医療的助言ではありません」
- 「心配な症状がある場合は、医師にご相談ください」
- 「個人の健康状態に応じて、医療専門家の指示を優先してください」

## 心理学的アプローチ

### 自己決定理論（Self-Determination Theory）

**原則**: 自律性、有能感、関係性の 3 つの基本的欲求を満たす表現。

**自律性（Autonomy）を支援**：

```
❌ 「毎日8時間寝てください」
✅ 「あなたに合った睡眠時間を見つけましょう。7-9時間が一般的です」
```

**有能感（Competence）を高める**：

```
❌ 「まだ目標に達していません」
✅ 「素晴らしい！先週より15%改善しています」
✅ 「小さな一歩ですが、確実に前進しています」
```

**関係性（Relatedness）を感じさせる**：

```
✅ 「一緒に健康的な習慣を作りましょう」
✅ 「あなたの健康をサポートします」
✅ 「同じ目標を持つ多くの人が〜を実践しています」
```

### ナッジ理論（Nudge Theory）

**原則**: 強制ではなく、選択の自由を保ちながら望ましい行動を促す。

**デフォルト設定の提示**：

```
✅ 「おすすめの設定を適用しました。いつでも変更できます」
✅ 「多くの人は朝7時にリマインダーを設定しています」
```

**社会的証明（Social Proof）**：

```
✅ 「同年代の80%が週3回以上運動しています」
✅ 「似た目標の人は平均〜を実践しています」
```

**損失回避（Loss Aversion）の適切な活用**：

```
❌ 「このままでは健康を損ないます」（恐怖訴求は避ける）
✅ 「今の良い習慣を継続しましょう」（ポジティブフレーミング）
```

### ポジティブ心理学

**原則**: 強みと成長に焦点を当て、ポジティブな感情を引き出す。

**成果の認識**：

```
✅ 「3日連続で達成です！素晴らしい」
✅ 「今週のハイライト：睡眠の質が向上しました」
✅ 「あなたの努力が実を結んでいます」
```

**未来志向**：

```
❌ 「昨日はできませんでしたね」
✅ 「明日は新しいチャンスです」
✅ 「次はもっと良くなります」
```

### 成長マインドセット（Growth Mindset）

**原則**: 失敗は学びの機会であり、能力は成長可能だという考え方。

**失敗時の対応**：

```
❌ 「目標を達成できませんでした」
✅ 「今回はうまくいきませんでしたが、次に活かせることが見つかりました」
✅ 「完璧でなくても大丈夫。小さな改善を積み重ねましょう」
```

## シーン別メッセージング

### 1. オンボーディング

**初回起動時**：

```
✅ 「TempoAIへようこそ！あなたの健康をサポートします」
✅ 「まずはあなたのことを教えてください。より良いアドバイスができます」
```

**権限要求時**：

```
❌ 「HealthKitへのアクセスを許可してください」
✅ 「より正確なアドバイスのため、HealthKitのデータを活用させていただけますか？」
✅ 「あなたのデータは安全に保護され、デバイス内のみで処理されます」
```

**設定完了時**：

```
✅ 「準備完了です！さっそく始めましょう」
✅ 「あなた専用の健康プランを作成しました」
```

### 2. 日々のアドバイス

**朝のメッセージ**：

```
✅ 「おはようございます！今日も良い一日を」
✅ 「昨夜はよく眠れましたか？今日のおすすめをご覧ください」
✅ 「今日の天気は〜です。屋外での運動はいかがでしょうか？」
```

**食事に関するアドバイス**：

```
❌ 「タンパク質が不足しています」
✅ 「今日はタンパク質を意識してみませんか？魚や豆類がおすすめです」
✅ 「昼食に〜を追加すると、栄養バランスが整います」
```

**運動に関するアドバイス**：

```
❌ 「運動してください」
✅ 「短い散歩から始めてみませんか？10分でも効果があります」
✅ 「今日は天気が良いですね。外で体を動かしてみましょう」
```

**睡眠に関するアドバイス**：

```
✅ 「就寝の1時間前です。リラックスタイムを作ってみませんか？」
✅ 「昨夜より30分早く眠れましたね。良い傾向です」
```

### 3. データフィードバック

**良好な結果**：

```
✅ 「素晴らしい！今週は目標を達成しました」
✅ 「先月より睡眠の質が15%向上しています」
✅ 「この調子で続けましょう。着実に改善しています」
```

**改善の余地がある結果**：

```
❌ 「悪化しています」
✅ 「今週は少し変動がありました。来週は〜を意識してみませんか？」
✅ 「ここ数日は睡眠時間が短めでした。今夜は早めに休んでみては？」
```

**データ不足時**：

```
❌ 「データがありません」
✅ 「より良いアドバイスのため、データを記録してみませんか？」
✅ 「今日の活動を記録すると、パーソナライズされた提案ができます」
```

### 4. 目標設定

**目標提案時**：

```
❌ 「目標を設定してください」
✅ 「あなたに合った目標を見つけましょう」
✅ 「小さな目標から始めるのがおすすめです」
```

**目標達成時**：

```
✅ 「おめでとうございます！目標達成です🎉」
✅ 「素晴らしい努力でした。次の目標はいかがですか？」
```

**目標未達成時**：

```
❌ 「目標に達しませんでした」
✅ 「もう少しでした！次回は達成できそうですね」
✅ 「今回の経験を活かして、次に進みましょう」
```

### 5. エラー・制約時

**データ取得エラー**：

```
❌ 「エラーが発生しました」
✅ 「データの更新中です。少々お待ちください」
✅ 「接続を確認しています。しばらくお待ちください」
```

**権限が不足**：

```
❌ 「権限がありません」
✅ 「より良い体験のため、設定から〜へのアクセスを許可していただけますか？」
✅ 「この機能を使うには、〜の権限が必要です。設定から変更できます」
```

**機能制限**：

```
❌ 「この機能は使えません」
✅ 「この機能は近日公開予定です。お楽しみに」
✅ 「現在開発中の機能です。リリースをお待ちください」
```

### 6. 通知・リマインダー

**定期リマインダー**：

```
❌ 「水を飲んでください」
✅ 「水分補給の時間です💧 体調はいかがですか？」
✅ 「そろそろ一息つきませんか？」
```

**モチベーション維持**：

```
✅ 「7日連続で達成中です。素晴らしい！」
✅ 「あなたならできます。今日も一歩前進しましょう」
```

### 7. レポート・サマリー

**週次レポート**：

```
✅ 「今週のハイライト」
✅ 「あなたの成長：先週と比べて〜が改善しました」
✅ 「来週の目標：〜を意識してみませんか？」
```

**月次レポート**：

```
✅ 「今月の振り返り：素晴らしい成果です」
✅ 「先月からの変化：〜が大きく改善しています」
✅ 「来月に向けて：〜に挑戦してみませんか？」
```

## 日本語表現のニュアンス

### 柔らかい提案表現

**「〜ませんか？」形式（推奨）**：

```
✅ 「試してみませんか？」
✅ 「始めてみませんか？」
✅ 「挑戦してみませんか？」
```

**「〜ますか？」形式（やや直接的）**：

```
△ 「試してみますか？」（Yes/Noの二択を迫る印象）
```

**「〜ましょう」形式（提案だが少し強め）**：

```
△ 「一緒に頑張りましょう」（使う場合は文脈に注意）
✅ 「一緒に目標を達成しませんか？」（より柔らかい）
```

### 丁寧度のバランス

**過度に丁寧すぎる（避ける）**：

```
❌ 「お試しいただけますでしょうか」（距離感が遠い）
```

**適切な丁寧さ**：

```
✅ 「試してみませんか？」
✅ 「いかがでしょうか？」
✅ 「おすすめです」
```

**カジュアルすぎる（場面による）**：

```
△ 「やってみて！」（親しみはあるが、健康アドバイスには不適切な場合も）
✅ 「やってみませんか？」（バランスが良い）
```

### 語尾の使い分け

**断定を避ける**：

```
❌ 「〜です」「〜ます」（命令的に聞こえる場合）
✅ 「〜かもしれません」「〜と良いでしょう」「〜がおすすめです」
```

**疑問形の活用**：

```
✅ 「〜してみては？」
✅ 「〜はいかがですか？」
✅ 「〜してみませんか？」
```

### クッション言葉

**提案前の緩衝**：

```
✅ 「もしよろしければ、〜」
✅ 「可能であれば、〜」
✅ 「お時間があるときに、〜」
```

**ネガティブ情報の緩和**：

```
❌ 「睡眠時間が短すぎます」
✅ 「少し睡眠時間が短めのようです」
✅ 「もう少し睡眠時間があると良いかもしれません」
```

## 英語表現のパターン

### 提案表現

**柔らかい提案**：

```
✅ "Would you like to try...?"
✅ "How about trying...?"
✅ "You might want to consider..."
✅ "It could be helpful to..."
```

**強すぎる表現（避ける）**：

```
❌ "You should..."
❌ "You must..."
❌ "You need to..."
```

### ポジティブフレーミング

**進捗重視**：

```
✅ "You're making progress!"
✅ "Great improvement from last week"
✅ "You're on the right track"
```

**欠点指摘（避ける）**：

```
❌ "Your sleep is insufficient"
❌ "You failed to meet your goal"
```

### エンパワーメント

**ユーザーの選択権**：

```
✅ "It's up to you"
✅ "You can adjust this anytime"
✅ "Choose what works best for you"
```

### 根拠提示

**データベース**：

```
✅ "Based on your data, ..."
✅ "Your patterns suggest..."
✅ "Research shows that..."
✅ "Many people find that..."
```

## 避けるべき表現

### 1. 恐怖訴求（Fear Appeal）

**❌ 絶対に避ける**：

```
「このままでは病気になります」
「危険な状態です」
「健康を損なう可能性があります」
"You're at risk of..."
"This is dangerous for your health"
```

**理由**:

- 不安を煽り、ストレスを増大させる
- 逆効果（防衛的反応を引き起こす）
- 医療的診断と誤解される

### 2. 過度な医療用語

**❌ 避けるべき**：

```
「高血圧のリスクがあります」
「糖尿病予防が必要です」
「心血管疾患の可能性」
```

**✅ 代替表現**：

```
「健康的な血圧を維持しましょう」
「バランスの良い食事を心がけませんか？」
「心臓の健康をサポートする習慣を作りましょう」
```

### 3. 絶対的表現

**❌ 避けるべき**：

```
「必ず効果があります」
「絶対に〜してください」
「これが唯一の方法です」
"This will definitely work"
"You must do this"
```

**✅ 代替表現**：

```
「多くの場合、効果が期待できます」
「〜がおすすめです」
「〜も選択肢の一つです」
"This often helps"
"This is recommended"
```

### 4. 比較・競争の煽り

**❌ 避けるべき**：

```
「他の人より遅れています」
「平均以下です」
「同年代に負けています」
```

**✅ 代替表現**：

```
「あなたのペースで進めましょう」
「過去の自分と比べて成長しています」
「自分らしい目標を設定しましょう」
```

### 5. 批判的・否定的表現

**❌ 避けるべき**：

```
「間違っています」
「失敗しました」
「できていません」
"That's wrong"
"You failed"
```

**✅ 代替表現**：

```
「別の方法を試してみませんか？」
「次はもっと良くなります」
「改善の余地があります」
"Let's try a different approach"
"You'll do better next time"
```

### 6. 一般化・決めつけ

**❌ 避けるべき**：

```
「すべての人に効果があります」
「あなたには〜が必要です」
「〜すべきです」
```

**✅ 代替表現**：

```
「多くの人に効果があります」
「〜が役立つかもしれません」
「〜を検討してみてはいかがでしょうか？」
```

## データに基づくフィードバック

### 定量的データの提示

**具体的で理解しやすい**：

```
✅ 「今週の平均睡眠時間：6時間30分（先週比+30分）」
✅ 「歩数：8,500歩（目標の85%）」
✅ 「水分摂取：1.8L（推奨範囲内）」
```

**曖昧な表現（避ける）**：

```
❌ 「睡眠時間が増えました」（どのくらい？）
❌ 「もう少し歩きましょう」（具体性がない）
```

### 傾向とパターンの説明

**分かりやすい言語化**：

```
✅ 「週末は平日より2時間長く眠る傾向があります」
✅ 「午後3時頃に活動量が低下しています」
✅ 「運動後は睡眠の質が20%向上しています」
```

### コンテキストの提供

**単なる数字ではなく意味を説明**：

```
✅ 「睡眠時間7.5時間は理想的な範囲です」
✅ 「1日1万歩は心臓の健康に効果的とされています」
✅ 「水分摂取量2Lは、あなたの体重と活動量から計算した推奨値です」
```

### 改善提案の具体性

**アクションにつながる提案**：

```
❌ 「もっと運動してください」
✅ 「通勤時に一駅分歩くと、目標達成できそうです」
✅ 「朝の10分ストレッチから始めてみませんか？」
```

## 実装ガイドライン

### メッセージングの一貫性

**トーン&マナー**：

- フレンドリーだが専門的
- サポート的で押し付けがましくない
- ポジティブで励まし重視
- 科学的根拠に基づきつつ親しみやすい

**視点**：

- 二人称「あなた」を使用（「ユーザー」ではない）
- AI は「サポート役」として振る舞う
- 対話的で一方的な指示は避ける

### A/B テストの観点

以下の要素は、ユーザーテストで最適化を検討：

- 提案表現（「〜ませんか」vs「〜はいかがですか」）
- 絵文字の有無と種類
- データ提示の詳細度
- モチベーションメッセージの頻度

### ローカライゼーション

**日本語と英語の文化的違い**：

- 日本語：より間接的、丁寧、婉曲的表現
- 英語：より直接的で OK、ただし提案形式は維持

**タイミングと頻度**：

- 日本：控えめな通知頻度が好まれる
- 英語圏：やや高頻度でも OK だが個人設定を尊重

## チェックリスト

新しいメッセージを作成する際は、以下を確認：

- [ ] ユーザーに選択権があることを示しているか？
- [ ] ポジティブで成長を促す表現か？
- [ ] 医療的助言と誤解される表現はないか？
- [ ] 恐怖訴求や否定的表現を避けているか？
- [ ] 具体的で実行可能なアドバイスか？
