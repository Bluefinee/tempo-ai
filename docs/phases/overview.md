## 概要

本ドキュメントは、Tempo AI の開発を 15 のフェーズに分割し、各フェーズの目的・成果物・依存関係を管理するものです。

**v4.0 での主な変更点**:

- **コンディション画面の大幅リニューアル**: 4 つの小カード → 24 時間サークル図+HRV 統合表示 + 要因マップ + AI の見立て
- **詳細画面の統合**: 5 画面 → 1 画面（あなたのパターン + リズムを整えるヒント）
- **ストレスメトリクス削除**: HRV に統合
- **AI プロンプト拡張**: `condition_insight` フィールド追加
- **フェーズ数削減**: 18 → 15（詳細画面統合による）
- **Phase 9-11 を調整フェーズに**: 実装済み機能を新仕様に適合

## Part D: コンディション画面

### Phase 12: コンディショントップ（リニューアル版）

**目的**: 新設計のコンディション画面を実装

**背景**:
コンディション画面は大幅にリニューアルされました：

| 旧仕様                                        | 新仕様                           |
| --------------------------------------------- | -------------------------------- |
| 4 つの小カード（睡眠、HRV、活動量、ストレス） | 24 時間サークル図 + HRV 統合表示 |
| 環境情報（1 行表示）                          | 要因マップ（睡眠・環境・活動）   |
| -                                             | リズム安定度表示                 |
| -                                             | AI の見立て                      |

**スコープ**:

1. **24 時間サークル図 + HRV 統合表示**

   - 外周: 過去 7 日間の就寝・起床時刻をプロット
   - 睡眠帯: 昨夜の睡眠時間を塗りつぶし表示
   - 中央上部: HRV 値（ms）を大きく表示
   - 中央下部: 7 日平均との差分（▲+9% など）
   - 下部テキスト: 就寝時刻・起床時刻
   - サークル描画コンポーネント実装

2. **リズム安定度表示**

   - ステータス: 良好 / やや不安定 / 不安定
   - インジケーター: ●●●○○ のような 5 段階表示
   - 説明文: 条件分岐による生成（metrics-spec.md 参照）

3. **要因マップ（3 要因）**

   - 睡眠: 貢献度 + 詳細テキスト（例: 7.2h / 深い睡眠 1.8h）
   - 環境: 負荷度 + 詳細テキスト（例: 午後から低気圧）
   - 活動: 貢献度 + 詳細テキスト（例: 昨日 4,200 歩）
   - 貢献度プログレスバー表示
   - 算出ロジック実装（metrics-spec.md セクション 9 参照）

4. **AI の見立て**

   - Backend から取得した `condition_insight` を表示
   - 3-5 文の統合解釈
   - ニックネームでの呼びかけ

5. **「詳しく見る」ボタン**
   - 詳細画面（Phase 13）への遷移

**成果物**:

- コンディション画面が新仕様で完成
- 24 時間サークル図が正しく描画される
- 要因マップの貢献度が正しく算出・表示される
- AI の見立てが表示される
- Product Spec v4.2 / UI Spec v3.2 に一致

**設計書**: `12-phase-condition-top.md`（新規作成）

**参照ドキュメント**:

### Phase 13: 詳細画面（1 画面統合版）

**目的**: コンディション詳細を 1 画面に統合して実装

**背景**:
旧仕様では 5 つの詳細画面（睡眠、HRV、リズム、活動量、ストレス）がありましたが、
新仕様では 1 画面に統合されます。

**スコープ**:

1. **「あなたのパターン」セクション**

   - 散布図コンポーネント実装
     - 縦軸: 翌朝の HRV（ms）
     - 横軸: 前日の睡眠時間（時間）
     - データ点: 過去 30 日分
     - 参照線: 閾値（例: 7 時間）を点線で表示
   - インサイト文の生成
     - パターン分析ロジック実装（metrics-spec.md セクション 10 参照）
     - 優先度順に分析（睡眠時間 →HRV、就寝時刻 →HRV、歩数 → 深い睡眠）
     - 差が 10%以上の場合のみ表示
   - 表示条件: 2 週間以上のデータ蓄積後のみ表示
   - データ不足時: 「まだパターンを分析中です」メッセージ

2. **「リズムを整えるヒント」セクション**

   - 条件分岐による定型文表示（AI は使用しない）
   - 最大 2 つまで表示
   - 条件と表示テキスト:
     - 就寝時刻のばらつきが大きい (SD > 60 分) → 「毎日同じ時間に寝ることを目指しましょう」
     - 起床時刻が遅い (平均 > 8:00) → 「起床時刻を少しずつ早めて、朝の光を浴びましょう」
     - 週末シフトあり (平日と 2 時間以上差) → 「週末も平日と同じ時間に起きる習慣をつけましょう」
     - 就寝が遅い (平均 > 24:00) → 「就寝を毎日 15-30 分ずつ早めてみましょう」
     - 特に問題なし → 「このリズムを維持できると良いですね」

3. **画面遷移**
   - コンディショントップからの遷移（右からスライドイン）
   - 戻るボタン / スワイプで戻る

**成果物**:

- 詳細画面が 1 画面で完成
- 散布図が正しく描画される
- インサイト文が条件に応じて正しく生成される
- ヒントが条件分岐で正しく表示される
- Product Spec v4.2 / UI Spec v3.2 / Metrics Spec v3.0 に一致
